SDK Capabilities

What the SDK does

Not a camera wrapper. A full visual intelligence pipeline — capture, compress, upload, analyze, notify — in a single integration.

videocam

Video Capture

High-quality recording with guided workflows, automated framing, and configurable quality/compression. Works offline with automatic background sync.

cloud_sync

Smart Upload

Adaptive compression, chunked uploads, and resumable transfers. Handles poor connectivity gracefully — essential for field and shop floor environments.

query_stats

VQS Integration

Captured video is automatically routed to Video Query Service for structured data extraction. Results stream back to your app in real time via webhooks or polling.

chat

Messaging

Built-in two-way video messaging between technicians and customers. AI-generated summaries, read receipts, and configurable approval workflows.

palette

White-Label

Full UI customization — your brand, your colors, your experience. Partners deploy TruVideo-powered features under their own identity.

security

Enterprise Security

SOC 2 compliant. BYOS architecture — video stays in your cloud. End-to-end encryption, role-based access, and complete audit trails.

Integration

Integrate once, deploy everywhere

Native iOS & Android

Swift and Kotlin SDKs with full platform API access. Embed as a module in your existing app or build standalone. Average integration time: 2-4 weeks to production.

Web SDK

JavaScript SDK for browser-based capture and viewing. Powers Send2Me contactless capture links — no app install required for end users.

REST & Webhook APIs

Full REST API for server-side integration. Webhooks for real-time event streaming — video uploaded, analysis complete, message delivered, approval granted.

Pre-Built Connectors

CDK Global, DealerSocket, Tekion, Reynolds & Reynolds in automotive. Open connector framework for any DMS, ERP, CMMS, or field service platform.
