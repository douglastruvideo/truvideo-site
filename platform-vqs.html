query_stats Core Technology

Query video like a database

Video Query Service (VQS) is TruVideo's core intelligence engine. It transforms unstructured video into structured, searchable, actionable data — damage types, severity scores, timestamps, text, objects, and conditions — all queryable via API.

How VQS Works

From raw footage to structured intelligence

upload_file

1. Ingest

Video arrives from any source — SDK, web upload, Send2Me link, API batch. VQS normalizes format, resolution, and encoding automatically.

psychology

2. Process

AI models analyze every frame: object detection, damage classification, severity scoring, OCR, scene segmentation, and temporal mapping. Running on BYOAI architecture — swap models without rewriting pipelines.

database

3. Structure

Results are stored as structured records with full provenance — timestamp, source frame, confidence score, model version. Query via API, stream via webhooks, or aggregate in dashboards.

Capabilities

What VQS extracts

Damage Detection & Classification

Scratches, dents, cracks, corrosion, fluid leaks — classified by type, severity, and location. Trained on 80M+ real-world inspection videos across multiple asset types.

Condition Scoring

Automated condition grades for assets, components, and surfaces. Consistent, repeatable, and calibrated against human expert assessments.

Temporal Attribution

Not just what damage exists — when it appeared. VQS builds asset memory by comparing inspections over time, creating verifiable condition timelines.

Text & Document Extraction

OCR for VIN plates, serial numbers, work orders, labels, and signage. Structured extraction feeds directly into DMS, ERP, and CMMS systems.
